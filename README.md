## Installation

### Gradle Plugin

The Gradle plugin is available at [Gradle Plugin Portal](https://plugins.gradle.org/).

Apply the Gradle plugin in the build script of your project:

####
```kotlin
plugins {
    id("io.github.manriif.supabase-functions") version "0.0.1"
}
```
